# 组件配置

通过配置文件的形式来配置 `k8s` 自有组件

### 分类

- 来源
    - 命令行标识
    - 序列化并存储在磁盘上的 `API` 类型
    - 序列化并存储在 `k8s API` 中的 `API`
- 分类
  - 引导配置
    - 组件启动依赖的配置，在组件启动前加载
  - 动态配置
    - 正常操作中会被更改的配置项
  - 静态配置
    - 后续组件部署，发布中被更改的配置项目
  - 共享配置
  - 独有配置
  - `Feature Gates`
    - 功能门，默认情况下被认为不安全而被禁用的配置项
  - 请求上下文
    - 某次请求有关的配置
  - 环境
    - 节点名等
